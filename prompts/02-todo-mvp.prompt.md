- 할 일 추가
- 완료 체크

- 필터

제약:

- 모바일 우선 UI
- 단순하고 깔끔한 디자인
- 상태 관리는 최소한으로

출력 형식:

1. 화면 구조
2. 핵심 컴포넌트
3. 코드
4. 실행 방법
